To view your IMDb statistics, this website will need a copy of your ratings. If you are worried about personal information the file only contains data about the media you have watched so no personal information is gotten.

Visit Your Ratings on IMDb
And make sure to log in if you aren't already. Click the three dots in the top right corner.
When you see the export button, click it and make sure to save the file as ratings.csv or else the website won't work.
